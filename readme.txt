== Description ==
Engage visitors by letting them pick politicians & compare their grass roots political popularity visually with this graph generator sidebar widget.
== Installation ==
1. Upload `wp-ppw.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Widgets (under Appearance) and drag "WP Political Popularity Widget" to the sidebar.
4. Open "WP Political Popularity Widget" in sidebar and fill in the title, the first politician's name and the second politician's name.
5. Save and close sidebar widget.
